# Notes App - Fullstack

A fullstack notes application built as a learning project with microservices architecture.

## Project Structure

This monorepo contains two services:

```
notes-app/
├── web-notes/     # Frontend - React + TypeScript + Vite
└── api-notes/     # Backend API (coming soon)
```

## Services

### Frontend (web-notes)

- **Tech Stack**: React, TypeScript, Vite
- **Purpose**: User interface for notes management

### Backend (api-notes)

- **Tech Stack**: TBD
- **Purpose**: API service for notes data
